# 🍰 CheeseyKek

A modern and aesthetic cheesecake shop website built using HTML, CSS, and JavaScript.

## ✨ Features

- Elegant bakery-style UI
- Responsive design
- Hero section with featured cheesecake
- About Us section
- Contact page
- Feedback page
- Soft pastel aesthetic
- Custom cheesecake branding

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

├── index.html
├── about.html
├── contact.html
├── feedback.html
├── style.css
├── main.js
├── image.png

## 🚀 How to Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser

## 🎨 Theme

Soft pastel dessert-inspired UI focused on a premium cheesecake brand experience.
